# 错误页风格模板工具 (Error Page Style Tools)

这是一个用于管理和切换 Nginx 错误页面风格的工具集。内置了多种种不同设计风格的精美模板，支持一键切换，会自动部署对应的 HTML 文件并更新引导页 (`index.html`)。

无需手动编辑代码，只需运行脚本即可快速应用心仪的错误页主题，提升网站的用户体验。

## 🚀 使用方法 (Usage)

运行根目录下的切换脚本：

```bash
python switch_style.py
```

脚本启动后会显示所有可用风格的菜单，输入对应的数字编号即可立即应用。

## 🛠️ 目录结构

* `switch_style.py`: 风格切换主程序
* `script/`: 存放各版本风格的定义与部署脚本
* `assets/`: 存放预览图片
* `*.html`: 部署后生成的错误页面文件（Nginx 可直接调用）
